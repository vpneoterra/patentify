GitHub Secrets Integration

For the vpneoterra/fusion-ip-whitespaces repo, add secrets via Settings → Secrets → Actions. The deploy workflow injects them as window.PATENTIFY_CONFIG variables at build time.

Repository Secrets:
ANTHROPIC_API_KEY → sk-ant-...
SUPABASE_URL → https://xyz.supabase.co
SUPABASE_ANON_KEY → eyJhbGci...

In deploy.yml:
echo "window.PATENTIFY_CONFIG={ANTHROPIC_API_KEY:'${{secrets.ANTHROPIC_API_KEY}}'};" > env.local.js
